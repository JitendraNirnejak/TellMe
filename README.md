# TellMe
Application to collect and analyze Irrigation Data.

### Version
0.2

## Usage

### Clone
Clone the Project

```sh
$ git clone https://github.com/JitendraNirnejak/TellMe.git
$ cd sass_boilerplate
```

### Installation

Install the dependencies (flask, psycopg etc.)

```sh
$ pipenv install
```

### Run

This is start the server

```sh
$ python app.py
```
